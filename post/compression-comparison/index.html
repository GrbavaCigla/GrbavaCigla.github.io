Compression Comparison - Wed, Dec 2, 2020

Here I am going to show differences in various compression algorithms. This is a handy cheatsheet for (de)compressing commands.

Compressing

Gunzip

tar -cjvf test.tar.gz test
  • c - create archive
  • j - gz
  • v - verbose
  • f - file name of output

Bzip2

tar -czvf test.tar.bz2 test
  • z - bz2

XZ

tar -cJvf test.tar.xz test
  • J - xz

Zip

zip -r test.zip test
  • r - recursive

7zip

7z a test.7z test
  • a - add

Rar

rar a test.rar test

Decompressing

Tar

tar -xfv test.tar.*
  • x - extract
  • f - file to extract
  • v - verbose

Zip

unzip test.zip

7zip

7z x test.7z

Rar

rar x test.rar

Source is available at: https://github.com/GrbavaCigla/CompressionComparison
